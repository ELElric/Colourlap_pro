# ColorLab Pro — 领域知识

> 此文件是色度科学（color science）领域知识的**最终裁定**。当其他文档与之冲突时，以此为准。

## 1. CIE 1931 XYZ

### 1.1 定义

- 使用 CIE 1931 2° 标准观察者色匹配函数（CMFs）：x̄(λ)、ȳ(λ)、z̄(λ)
- 波长范围 360–830 nm，1 nm 步长
- 通过 `colour-science` 的 `sd_to_XYZ` 计算

### 1.2 计算公式

```
X = k · ∫ S(λ) · x̄(λ) dλ
Y = k · ∫ S(λ) · ȳ(λ) dλ
Z = k · ∫ S(λ) · z̄(λ) dλ
```

其中 k 是归一化常数。

## 2. CIE 1931 xy 色度

```
x = X / (X + Y + Z)
y = Y / (X + Y + Z)
```

- D65 白点：(0.3127, 0.3290)
- 等能白 E：(1/3, 1/3)

## 3. CIE 1976 u-prime-v-prime

```
u'' = 4x / (-2x + 12y + 3)
v'' = 9y / (-2x + 12y + 3)
```

- 更接近感知均匀

## 4. McCamy CCT

```
n = (x - 0.3320) / (0.1858 - y)
CCT = 437·n³ + 3601·n² + 6861·n + 5517
```

- 适用范围：3000–50000 K
- 在 0.3320 ≤ x ≤ 0.1858y + ... 区域内有效（详见 McCamy 1992 原文）

## 5. Lambert-Beer 定律

```
T(λ) = 10^(-α(λ) · d)
```

- T：透射率
- α(λ)：吸收系数（与波长有关）
- d：膜厚

## 6. Coverage / Match

- **Coverage**：目标多边形与设备多边形的交集面积 / 目标面积
- **Match**：(1 − mean(Δxy) / 0.1) × 100%
- 几何运算使用 shapely（在 xy 空间）

## 7. 光谱标准

- 波长 380–780 nm，1 nm 步长，401 点
- 单位：`mW/nm`、`counts`、`a.u.`、`transmittance`
- 数值精度：float64

## 8. 色度空间选择

- **所有 Coverage / Match 在 xy 空间**（D-016）
- 显示对比、可视化容易、业界标准
