# 02 UI 布局规范

> 状态：与 01 一致（Frozen for V1.1）

## 1. 主窗口

- 默认分辨率：1440×900
- 主窗口由 QMainWindow 承载，中心是 QStackedWidget（页面切换）
- 左侧 QListWidget（导航），宽度 200px
- 右侧 QDockWidget（属性面板），宽度 320px，可关闭

## 2. 页面布局

每个工作区页面统一采用：

```
+----------------------------------+
| PageTitle (H1)         [Actions] |
+----------------------------------+
| [Toolbar]                         |
+----------------------------------+
| [Workspace]                       |
|                                   |
+----------------------------------+
| [StatusBar: 光谱点数 / 单位 / 状态]|
+----------------------------------+
```

## 3. 关键交互

- **拖拽导入**：所有光谱列表区支持拖入 .csv / .xlsx
- **右键菜单**：光谱项提供 Duplicate / Rename / Delete / Export
- **多选**：Ctrl / Shift 多选，批量操作（删除 / 导出 / 颜色标记）
- **图表缩放**：鼠标滚轮 / 框选 / 双击复位
- **单位切换**：所有色度显示支持 xy ↔ u-prime-v-prime ↔ XYZ

## 4. 主题

- 暗色默认
- 字体：Microsoft YaHei UI（中文）/ Segoe UI（英文）
- 色板：背景 #1E1E1E，前景 #E0E0E0，强调 #4FC3F7
- 详见 `resources/styles/dark.qss`（T-12 阶段产出）
