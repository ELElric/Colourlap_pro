# 09 参考数据集规范

> 内置的、随包发布的参考数据。

## 1. CIE 1931 2° 标准观察者

- 来源：内置于 `colour-science` 包
- 波长范围 360–830 nm，1 nm 步长
- 使用方式：`colour.colorimetry.MSDS_CMFS["CIE 1931 2 Degree Standard Observer"].values`

## 2. 标准光源

| 名称 | 说明 |
|------|------|
| D65 | CIE 标准日光 |
| D50 | CIE 标准日光 |
| A | CIE A 光源（白炽灯） |
| C | 已废弃，保留参考 |
| E | 等能白光 (1,1,1) |

## 4. 参考光谱（合成）

随包提供三组测试光谱（`resources/reference_data/synthetic/`）：

- `led_r.csv`：典型 R-LED（峰值 630 nm，FWHM 20 nm）
- `led_g.csv`：典型 G-LED（峰值 530 nm，FWHM 30 nm）
- `led_b.csv`：典型 B-LED（峰值 450 nm，FWHM 20 nm）

每组 401 点，380–780 nm，1 nm 步长。

## 5. 数据文件格式

CSV：
```csv
wavelength_nm,intensity
380.0,0.000123
381.0,0.000145
...
```

XLSX：第一列 `wavelength_nm`，第二列起为不同光谱。
